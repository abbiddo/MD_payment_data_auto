# payment-data-automation

pandas와 Google Sheet를 활용한 결제 데이터 자동화 프로젝트입니다.

반복적으로 진행하던 결제 내역 정리 및 시트 업데이트 작업을 자동화하기 위해 제작했습니다.

---

## Background

온라인 교육 서비스 운영 중 다음과 같은 반복 업무가 존재했습니다.

- 결제 데이터 필터링
- 상태값 분류 및 정리
- 담당자별 데이터 분리
- Google Sheet 수동 업데이트
- 주차별 데이터 정리

기존에는 시트에서 직접 필터링 및 정리를 반복적으로 수행했으며,
반복 작업에 많은 시간이 소요되었습니다.

이를 개선하기 위해 pandas 기반 자동화 코드를 작성했습니다.

---

## Tech Stack

- Python
- pandas
- numpy
- gspread
- Google Colab

---

## Workflow

1. 결제 데이터 CSV 불러오기
2. 조건 기반 데이터 정제
3. 상태값 자동 분류
4. 시트 출력용 데이터 생성
5. 담당자별 데이터 분리
6. Google Sheet 자동 업로드

---

## Sample Data

실제 업무 데이터는 포함하지 않았으며,
구조를 재현한 샘플 데이터를 사용했습니다.

---

## Result

기존 수작업 기반 데이터 정리 과정을 자동화하여
기존 약 30분 소요되던 수작업을 5분 이내로 단축하고 데이터 관리 흐름을 단순화했습니다.

정제된 데이터는 Google Sheet에 자동 업로드되며,

- 전체 데이터 탭
- 담당자별 탭

형태로 분리 저장됩니다.

---

## Note

실제 업무 데이터를 기반으로 작성되었으며,
